What is Docker

Docker is a tool that helps in the creation, deployment, and running of applications by the use of containers.A container is a package that contains all needed libraries and dependencies. This allows a developer to release a package of code that can be run on any Linux setup. Source: https://opensource.com/resources/what-docker

What is Nginx

Nginx is an open source software that is used for web serving, reverse proxying, caching, load balancing, and media streaming. Nginx was developed to help speed up web servers with large concurrent connections. Source: https://www.nginx.com/resources/glossary/nginx/

What is VM (virtual machine)

A virtual machine is a computer file that behaves like its own, independent computer. The virtual machine is sandboxed, which means that it is isolated and does not affect the computer you are connecting to it from. The virtual machine creates an ideal environement to test software or to backup software and data. Source: https://azure.microsoft.com/en-us/overview/what-is-a-virtual-machine/

Who is Cloud Computing Service Provider - Digitalocean

DigitalOcean is a company that offers Infrastructure as a Service (IaaS). DigitalOcean lets you create a virtual machine that can be run on the cloud. They compete against services like Amazon Web Services and the Google Compute Engine. They have a variety of different virtual machine sizes to choose from, and you can host these on different servers across the world. Source: https://searchcloudcomputing.techtarget.com/definition/DigitalOcean
